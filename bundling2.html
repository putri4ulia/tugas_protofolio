PAKET 2


PENYELESAIAN

Persamaan Awal:
(1) x + 2y + z = 155.000
(2) x + y + 2z = 180.000
(3) 2x + y + 2z = 205.000
1) Eliminasi Persamaan (1) dan (2)
x + 2y + z = 155.000
x + y + 2z = 180.000

Dikurangkan → -y + z = 25.000→ (4)
2) Eliminasi Persamaan (2) dan (3)
x + y + 2z = 180.000
2x + y + 2z = 205.000

Dikurangkan → 205-180

x = 25 → (5)
3) Eliminasi Persamaan (4) dan (5)
z-y = 25

Dikurangkan → z= y + 25 → (6)
4) Substitusi ke persamaan (1)
x + 2y + z = 155.000
25.000 + 2y + (y + 25.000) = 155.000
25.000 + 2y + y + 25.000 = 155.000
3y + 50.000 = 155.000
3y = 105.000

y = 35.000
5) Substitusi y ke persamaan (6)
z = y + 25.000
z = 35.000 + 25.000

z = 60.000
✔ x = 25.000 (Harga tas laptop)
✔ y = 35.000 (Harga tas tutup botol plastik)
✔ z = 60.000 (Harga tas sampah plastik kopi)
